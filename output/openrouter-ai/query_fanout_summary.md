## 核心定位

OpenRouter 是一个**统一 AI API 网关**，通过单一 OpenAI 兼容端点提供 400+ AI 模型的接入，覆盖 50+ 供应商，充当开发者的"模型路由中枢"。

## 关键信息汇总

### 平台能力

| 功能 | 详情 |
|------|------|
| 模型数量 | 400+ AI 模型 |
| 供应商 | 50+ 供应商（OpenAI、Anthropic、Google、Meta 等） |
| API 兼容 | OpenAI SDK 兼容，零代码改动迁移 |
| 智能路由 | Pareto Router、Auto Router、Free Models Router |
| 回退机制 | 自动供应商切换、故障转移 |
| 多模态 | 文本、图像生成、视频生成 |
| Embedding | 22+ 嵌入模型（2025年11月上线） |

### 定价策略

- **穿透定价（Pass-through pricing）**：模型费率无加价
- **平台费**：信用卡充值收取 5.5%
- **免费模型**：29 个免费模型，零成本实验
- **付费模式**：按量付费（Pay-as-you-go），充值为余额制
- **企业计划**：提供 Enterprise 定制方案

### 核心特性

1. **统一接口**：同一代码结构访问 GPT-4o、Claude、Gemini、Llama 等数百个模型
2. **智能路由**：自动选择最优模型（按价格/延迟/吞吐量）
3. **成本效率**：透明按 token 计费，数十个免费模型
4. **高可靠性**：内置自动回退和故障转移
5. **多模态支持**：文本、图像、视频生成模型
