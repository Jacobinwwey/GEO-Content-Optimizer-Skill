## 查询 3: linux.do 注册 用户数 信任等级 福利

### 搜索结果

1. **注册方式**
   - 邀请注册制（曾开放过公开注册）
   - 2025年起取消了强制邀请码限制
   - 新用户申请需填写 50字左右的真实自述
   - 3级用户可创建邀请，每日最多1个，邀请人需大陆 IP

2. **信任等级（Trust Level）**

   | 等级 | 名称 | 说明 |
   |------|------|------|
   | 0级 | 新用户 (New) | 刚注册初始状态 |
   | 1级 | 基本用户 (Basic) | 需要一定活跃度 |
   | 2级 | 成员 (Member) | 60分钟阅读、50%天数访问、10+话题回复 |
   | 3级 | 活跃用户 (Regular) | 100天内长期活跃 |
   | 4级 | 领导者 (Leader) | 最高信任等级 |

3. **福利体系**
   - hub.linux.do：积分兑换服务（如 GPT-5.5 等）
   - "福利羊毛"版块
   - 不定期抽奖活动

## 查询 5: linux.do 评价 争议 知乎

### 搜索结果

1. **知乎争议**
   - 知乎问题"如何评价 LINUX DO 社区？"引发讨论
   - 批评：管理权限集中，封禁过于随意
   - 批评：与 Linux 开放共享精神不符

2. **产业调查**
   - UniFuncs 发布深度调查报告， 
   - 是论坛最具争议性的外部评价之一

3. **正面评价**
   - 部分用户称其为互联网"桃花源"
   - 认为社区氛围友善，是简中互联网"清流"

4. **注册争议**
   - 有用户反映入站申请被无视
   - 审核不透明引发讨论
