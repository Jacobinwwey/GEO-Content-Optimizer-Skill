# 查询扩展摘要 — 302.AI

**分析网址**: https://302.ai/
**生成日期**: 2026-05-10

---

## 核心定位

302.AI 是一个**企业级 AI 资源聚合平台**，以"AI 超市"为理念，提供按用量付费的多模型 API 接入和开箱即用的 AI 应用。

## 关键信息汇总

### 平台架构

| 板块 | 功能 | 目标用户 |
|------|------|----------|
| 机器人超市 | 预构建 AI 机器人（客服、写作、编程、翻译） | 非技术用户 |
| 工具超市 | 在线 AI 工具（图片生成、视频处理、文档转换） | 全部用户 |
| API 超市 | 统一 API 接口，1400+ API 接入 | 开发者/企业 |

### 模型覆盖

- **国际模型**: OpenAI GPT 系列、Anthropic Claude 系列、Google Gemini 系列、xAI Grok 系列
- **国内模型**: DeepSeek、通义千问（Qwen）、智谱 GLM、百川、月之暗面
- **多模态**: 图像（DALL-E、Midjourney、Stable Diffusion）、语音（Whisper、TTS）

### 定价策略

- 按用量付费，无最低消费、无月费
- 无 TPM（Token Per Minute）限制
- 价格示例：grok-4.3 $1.25/$2.5 per 1M tokens，chat-latest $5/1M tokens

### 竞争格局

- **302.AI** vs **OpenRouter**: 302.AI 提供应用层，OpenRouter 纯 API 路由
- **302.AI** vs **APIMart**: 302.AI 偏应用平台，APIMart 偏开发者基础设施
- **302.AI** vs **Together AI**: 302.AI 聚合多供应商，Together AI 聚焦开源模型

### 开源生态

- 30+ 深度开发的在线工具应用，每周新增 2-3 个
- 所有工具全部开源，支持 B 端私有化部署
- GitHub 上有活跃的开源社区

### 用户评价

- 中文用户群体反馈积极，降低了使用海外 AI 模型的门槛
- API 稳定性受认可
- 定价透明，试错成本低
