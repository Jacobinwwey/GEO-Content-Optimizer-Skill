# 查询扩展研究 — 302.AI

**页面标题**: 企业级AI资源平台 - 302.AI | 按用量付费，全模型API接入，应用在线使用
**分析网址**: https://302.ai/

---

## 查询 1: 302.AI 企业级AI资源平台

### 搜索结果

1. **302.AI - 企业级AI资源平台**
   - 302.AI 是一个面向企业和开发者的 AI 资源聚合平台
   - 三大核心板块：机器人超市、工具超市、API超市
   - 按用量付费模式，无需订阅
   - 支持全模型 API 接入，包括 OpenAI、Anthropic、Google Gemini、Grok、DeepSeek、通义千问等
   - 强调无 TPM 限制、企业级稳定性

2. **APIMart vs 302.AI 对比**
   - APIMart 更偏向开发者基础设施
   - 302.AI 定位为应用平台，提供可直接使用的在线应用
   - 两者在 API 聚合领域有重叠但侧重点不同

---

## 查询 2: 302.AI API 价格 模型支持

### 搜索结果

1. **302.AI 定价模式**
   - 按用量付费（Pay-as-you-go），无最低消费
   - 支持主流大模型 API：GPT-5.5、Claude、Gemini 2.5、Grok 4.3、DeepSeek R1 等
   - 价格示例：grok-4.3 $1.25/$2.5 per 1M tokens，gpt-5.5 $5/$30 per 1M tokens
   - 无 TPM（Token Per Minute）限制，适合高并发场景

2. **模型覆盖范围**
   - 国际模型：OpenAI GPT 系列、Anthropic Claude 系列、Google Gemini 系列、xAI Grok 系列
   - 国内模型：DeepSeek、通义千问（Qwen）、智谱 GLM、百川、月之暗面等
   - 图像模型：DALL-E、Midjourney、Stable Diffusion
   - 语音模型：Whisper、TTS 系列

## 查询 3: 302.AI 机器人超市 工具超市

### 搜索结果

1. **机器人超市**
   - 提供预构建的 AI 机器人，涵盖客服、写作、编程、翻译等场景
   - 用户可直接使用，无需编程

2. **工具超市**
   - 在线 AI 工具集合：图片生成、视频处理、文档转换、数据分析等
   - 即开即用，按次计费
   - 部分工具开源，可在 GitHub 上获取

3. **开源生态**
   - 302.AI 开源了多个 AI 应用项目
   - GitHub 上有活跃的开源社区
   - 鼓励开发者贡献和二次开发

## 查询 4: AI API 聚合平台对比 2025 2026

### 搜索结果

1. **主要 AI API 聚合平台**
   - **302.AI**: 应用导向，机器人+工具+API 三合一
   - **OpenRouter**: 纯 API 路由，模型选择最广
   - **APIMart**: 开发者基础设施，注重稳定性和文档
   - **AI21 Labs**: 自研模型为主
   - **Together AI**: 开源模型托管和推理

2. **行业趋势**
   - AI API 聚合成为新赛道，降低企业接入多种模型的门槛
   - 按用量付费成为主流定价模式
   - 企业用户对稳定性、合规性要求提升
   - 国内平台开始提供国际模型的代理访问
